# agentic-news
### 你的 AI 新闻管家，不止聚合，更做认知升级

每天清晨，agentic-news 自动从高密度 RSS 源生成一份移动端 H5 晨报。
它不仅告诉你「发生了什么」，也解释「为什么重要、如何判断、接下来学什么」。

---

## 核心价值

- **高信号筛选**：每天 10–20 条真正值得读的精选
- **深度分析**：摘要、观点、风险/机会、反直觉观察
- **强个性化**：显式反馈 + 隐式行为双轨学习
- **学习建议**：每天给出可执行的认知和知识升级方向
- **移动优先体验**：干净、简洁、日报化阅读

---

## 工作流程

1. 抓取 RSS（科技 / 实政 / 财经等）
2. 清洗与去重
3. AI 分层分析（事实层 → 观点层 → 个性化建议层）
4. 综合评分排序（重要性、相关性、可信度、新颖度、时效性）
5. 生成 H5 日报并按 `YYYY/MM/DD` 归档
6. 通过 SFTP 上传到 Nginx，刷新 `/latest`
