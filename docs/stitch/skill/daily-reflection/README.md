# 🏛️ The Oracle's Daily Mirror (@agentic-news/daily-reflection)

Turn your OpenClaw agent from a simple coding assistant into your personal **Chief Strategy Officer**. 

While primitive skills like `self-improving-agent` only track coding errors and syntax typos, this overarching skill extracts high-level cognitive telemetry, calculates your focus entropy, and builds a comprehensive daily reflection of your mental architecture.

## 🌟 The Core Value Proposition
1. **Defend Your Attention**: Securely scans your local Git pacing and shell rhythms to generate an "Entropy vs Flow" baseline score.
2. **Cognitive Synthesis**: Transforms a disorganized mess of daily logs into an elegant, four-part philosophical briefing (Activity, Friction, Philosophy, Directive).
3. **100% Decentralized Sovereignty**: No APIs, no cloud sync, no tracking. Your telemetry never leaves your device. Builds a permanent, standalone "Digital Sanctuary".
4. **Seamless ClawHub Ecosystem**: Completely backwards-compatible with standard OpenClaw hooks, functioning entirely on your local machine.

## 🚀 Installation & Setup

Simply pull the package to your machine via ClawHub:
```bash
clawdhub install @agentic-news/daily-reflection
```

*(If utilizing directly via Git bare metal, clone this repo to `~/.openclaw/skills/daily-reflection`.)*

## 🧠 Workflow (How it operates securely)
1. **Trigger**: The skill binds to your end-of-day cron, or you simply tell your OpenClaw agent "Run my daily reflection."
2. **Context Intake**: It executes the local `activity-extractor.js` and `attention-analyzer.js` scripts safely inside your host environment.
3. **Synthesis**: The LLM processes your footprint to locate deep logic flaws and structural insights, automatically promoting the best mental models to your `SOUL.md`.
4. **Delivery**: An exquisite, highly-motivating Markdown report is delivered directly to `.openclaw/workspace/reflections/`.
